## [taotisi-cli](https://www.npmjs.com/package/taoists-cli)

taotisi-cli 是一个通过 npm 命令行进行项目构建的脚手架工具，通过该命令可以构建 vue 模板项目或者 react 模板项目。

## 下载安装

    安装
    npm i taoists-cli -g

    构建
    taoists-cli init
